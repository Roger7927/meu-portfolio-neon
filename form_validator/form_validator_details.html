Validador de Formulário Em Construção Contínua

Este projeto é um **Validador de Formulário** interativo, desenvolvido com HTML, CSS e JavaScript puro. Seu objetivo principal é garantir que os dados inseridos pelos usuários em um formulário de registro atendam a critérios específicos de validação em tempo real, fornecendo feedback claro e instantââneo.

É uma demonstração prática de como criar uma experiência de usuário mais robusta e amigável no front-end, evitando o envio de dados inválidos para o servidor e melhorando a usabilidade geral.

Tecnologias Utilizadas:

  • HTML5
  • CSS3 (Design Responsivo)
  • JavaScript (ES6+)
  • Regex (Expressões Regulares para Email)
  • Manipulação do DOM
  • Event Handling

Funcionalidades Principais:

  • Validação de Campos Obrigatórios
  • Verificação de Formato de Email
  • Validação de Comprimento Mínimo e Máximo para Nome de Usuário e Senha
  • Confirmação de Senha (verificação de correspondência)
  • Feedback Visual (bordas e mensagens de erro/sucesso)
  • Prevenção de Envio de Formulário Inválido

Desafios e Aprendizados:

  • Gerenciamento de Estado de UI: Desenvolver um sistema robusto para adicionar/remover classes CSS dinamicamente com base no estado de validação de cada campo.
  • Expressões Regulares: Aprofundar o conhecimento em Regex para criar validações de formato de email precisas e flexíveis.
  • Modularização do Código: Organizar o JavaScript em funções reutilizáveis para cada tipo de validação (obrigatório, comprimento, email, etc.).
  • Experiência do Usuário (UX): Focar em fornecer feedback claro e imediato ao usuário, tornando o processo de preenchimento do formulário mais intuitivo e menos frustrante.
